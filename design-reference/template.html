Coverage

Every tab. Designed.

The dashboard has 18 sub-tabs across 5 top-level destinations. This is the visual brief for each one.

01 / WORKSPACE
Overview

Mission status. Hero treatment with date in italic serif, 4 metric tiles with count-up, pipeline status, marquee of recent activity.

02 / WORKSPACE
Briefing

Editorial. Single-column centered body text in display type, 6 quick-action cards, recent activity as vertical timeline.

03 / WORKSPACE
Agents

6 agent cards in a row, 3 stat cards, large activity heatmap, collapsible "Run agent" panel on the right.

04 / WORKSPACE
Chat

Distraction-free. Centered message stream max-width 800px, assistant messages in glass cards, sticky input at bottom.

05 / STUDY
Upload

Hero dropzone with animated pipeline diagram, subject picker, options toggles, recent runs with progress bars.

06 / STUDY
Library · Notes

3-pane reading. Subject sidebar, note list with search, markdown reader with sticky TOC.

07 / STUDY
Library · Lecture Notes

Media-first. Lecture grid, vertical timeline with topic segments, synchronized transcript pane.

08 / STUDY
Research

Editorial + operational. New research form, history as cards, findings view with claims and sources.

09 / STUDY
Practice · Quiz

Focused. One question at a time, 4 option cards, instant feedback with correct/incorrect glow, explanation panel.

10 / STUDY
Practice · Flashcards

3D card flip on click/space, 4 difficulty buttons, deck info, current streak.

11 / PLAN
Schedule

Calendar-table elegant. 3 tables: Today's classes, Weekly schedule, Exams. Today highlighted with gradient bar.

12 / PLAN
Tasks

Kanban. 3 columns with italic-serif headers, draggable cards, drop targets glow on hover, color tags.

13 / PLAN
Focus

Meditative. 2-pane: large Pomodoro ring with mode color, 6 draggable sticky notes.

14 / KNOWLEDGE
Notebook

Card grid of notebooks. Name, source count, last activity, health indicator. Health banner at top.

15 / KNOWLEDGE
Notebook · Chat

RAG Q&A. Message stream with clickable citation chips, scope toggles in sidebar, source panel.

16 / KNOWLEDGE
Obsidian · Vault

3-pane. Vault tree, file list, markdown reader with internal link highlighting, backlinks.

17 / KNOWLEDGE
Obsidian · Search

Search-first. Large display-type input, results list with highlighted match line, recent searches empty state.

Philosophy

Five pillars. One language.

Pillar 01
Aurora background

Living background. Mesh gradient + drifting orbs + grain. Never a flat color. The background is the brand.

Pillar 02
Glassmorphism

Every surface is glass. Backdrop blur, low-opacity backgrounds, hairline borders. Nothing is solid.

Pillar 03
Editorial typography

Two type families. Fraunces italic for emphasis. Inter for everything else. Generous, breathing whitespace.

Pillar 04
Scroll-triggered motion

Content reveals as you scroll. IntersectionObserver + ease-out-expo. Counters animate. Orbs parallax.

Pillar 05
Micro-interactions

Every interactive element responds. Hover, focus, active. No dead UI. Magnetic buttons, gradient rings, pulsing dots.

Constraint
No build step

Alpine.js + vanilla CSS. Self-hosted fonts. Zero runtime CDN. Premium feel without premium tooling.
